description: 開発環境をセットアップ
---

このプロジェクトの開発環境をセットアップします。

以下の手順を実行してください：

1. 必要なディレクトリ構造を作成
   - flask-app/ (Flask管理GUI)
   - powershell-scripts/ (Windows自動化スクリプト)
   - drbl-server/ (DRBL設定ファイル)
   - tests/ (テストコード)
   - docs/ (ドキュメント)

2. Pythonプロジェクト初期化
   - requirements.txt 作成
   - Flask, SQLAlchemy, pytest等の依存関係定義

3. Git初期化（未初期化の場合）
   - .gitignore 作成
   - 初回コミット

4. README.md 更新
   - プロジェクト概要
   - セットアップ手順
   - 使用方法

各ステップ完了後、進捗を報告してください。
