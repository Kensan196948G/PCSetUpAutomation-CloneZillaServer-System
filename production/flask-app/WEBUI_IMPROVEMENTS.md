# Web UI Improvements - PC Setup Automation

このドキュメントでは、PC Setup Automation管理WebアプリケーションのUI/UX改善内容をまとめています。

## 改善概要

モダンなWebデザインとユーザビリティを向上させるため、全面的なUI/UXの刷新を実施しました。

## 主な改善点

### 1. ナビゲーション改善

**ファイル**: `templates/base.html`

#### 追加機能
- ドロップダウンメニュー方式の階層的ナビゲーション
- データインポートメニュー
  - CSV一括登録
  - ODJアップロード
- 展開管理メニュー
  - マスターイメージ管理
  - 展開設定
  - 展開ステータス
- アクティブページハイライト
- Bootstrap Icons統合
- モバイルレスポンシブ対応

### 2. ダッシュボード刷新

**ファイル**: `templates/index.html`

#### 新デザイン要素
- 統計情報カード（4種類）
  - 登録PC数
  - セットアップ完了
  - 進行中
  - 失敗
- アイコン付きビジュアルカード
- クイックアクションボタン
- 最新ログのタイムライン表示
- 最近登録PCのテーブル表示
- レスポンシブグリッドレイアウト

### 3. CSV一括登録UI

**ファイル**: `templates/csv_import.html`

#### 機能
- ドラッグ&ドロップファイルアップロード
- ビジュアルフィードバック（ドラッグオーバー時）
- プログレスバー表示
- ファイルプレビュー
- アップロード成功・失敗表示
- CSVフォーマット例の表示
- テンプレートダウンロード機能
- アップロードのポイント説明

**ルートハンドラー**: `views/import_export.py`
- `GET /csv-import`: アップロード画面表示
- `POST /csv-import/process`: CSVファイル処理

### 4. ODJファイルアップロードUI

**ファイル**: `templates/odj_upload.html`

#### 機能
- PC選択ドロップダウン
- 選択PC情報の表示
- ドラッグ&ドロップファイルアップロード
- ファイル形式バリデーション（.txt, .blob）
- ODJ説明とファイル形式ガイド
- セキュリティ警告表示

**ルートハンドラー**: `views/import_export.py`
- `GET /odj-upload`: アップロード画面表示
- `POST /odj-upload/process`: ODJファイル保存とPC紐付け

### 5. 展開ステータスダッシュボード

**ファイル**: `templates/deploy_status.html`

#### 機能
- リアルタイム進捗モニタリング
- サマリーカード（展開予定、完了、進行中、失敗）
- 全体進捗バー
- PC別カード表示
  - ステータスバッジ
  - 進捗バー（進行中の場合）
  - セットアップステップ表示
- フィルター機能
  - ステータス別
  - PC名/Serial検索
- 自動更新機能（5秒間隔）

**ルートハンドラー**: `views/deployment.py`
- `GET /deploy-status`: ステータスダッシュボード表示

### 6. マスターイメージ管理

**ファイル**: `templates/image_management.html`

#### 機能
- 登録済みイメージ一覧
- イメージ詳細情報（名前、パス、説明、サイズ、作成日）
- アクティブイメージ表示
- イメージ操作（編集、展開、削除）
- 新規イメージ登録モーダル
- DRBLサーバー情報表示
- クイックガイド

**ルートハンドラー**: `views/deployment.py`
- `GET /image-management`: イメージ管理画面表示

### 7. 展開設定

**ファイル**: `templates/deploy_settings.html`

#### 設定項目
- Clonezilla展開設定
  - 圧縮形式（zstd, gzip, lz4, none）
  - 展開モード（マルチキャスト、ユニキャスト）
  - 最大同時展開台数
  - タイムアウト設定
  - 自動再起動
  - イメージ整合性チェック
- PXEブート設定
  - PXEブートタイムアウト
  - デフォルトブート選択
  - PXEメニュータイトル
- ネットワーク設定表示（読み取り専用）

**ルートハンドラー**: `views/deployment.py`
- `GET /deploy-settings`: 展開設定画面表示
