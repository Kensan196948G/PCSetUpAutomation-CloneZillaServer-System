キッティング展開 QAチェックリスト

このチェックリストは、新規PCの自動展開（PXE〜ODJ〜後処理）が正常に機能しているかを確認するための項目である。

1. PXE/Clonezilla関連

項番 カテゴリ チェック項目 期待結果 結果 メモ
PXE-01 PXE 対象PCがPXEブートメニューまで自動で到達する DHCPからブート情報が返却され、Clonezillaが起動する
PXE-02 Clonezilla 指定したマスターイメージが正しく選択・展開される エラーなくイメージ書き込みが完了する
PXE-03 性能 10台同時展開時の所要時間を計測する 単体展開と比較して、時間増加が20%以内

2. Windows初期起動・PC名/ODJ関連

項番 カテゴリ チェック項目 期待結果 結果 メモ
WIN-01 初期起動 Windows初回起動時に自動ログオンが行われる ユーザー操作なしでスクリプトが起動する
WIN-02 Serial取得 スクリプトでSerial番号が正しく取得できる 箱側のSerialと一致している
WIN-03 PC名設定 PC名が命名規則（YYYYMMDDM）通りに設定される ホスト名が意図したPC名になっている
WIN-04 ODJ ODJファイル適用後に再起動し、ドメイン参加済みになる AD上でコンピュータアカウントが確認できる

3. 後処理（Windows Update / アプリ導入）

項番 カテゴリ チェック項目 期待結果 結果 メモ
POST-01 Windows Update スクリプトによる更新実行が完了する 再起動を含め、保留中の更新が無い状態になる
POST-02 アプリ 会社標準アプリがすべてインストールされている アプリ一覧で必要なアプリが揃っている
POST-03 OneDrive 次回ユーザーログオン時にOneDrive KFMが有効になる ドキュメント・デスクトップがOneDrive配下にリダイレクトされる

4. 管理GUI / ログ関連

項番 カテゴリ チェック項目 期待結果 結果 メモ
GUI-01 一覧 管理GUIでSerial/PC名/ステータスが一覧表示できる 対象PCがcompletedとして表示される
GUI-02 ログ 展開失敗時のエラー内容がログとして確認できる status=failed時に原因メッセージが残っている
