About ODJ Files

ODJ (Offline Domain Join) files are used to join Windows PCs to a domain without network connectivity during setup.

File Requirements:
  • Format: .txt or .odj
  • Max size: 10MB
  • Generated by: djoin.exe
How to generate ODJ file:
djoin /provision /domain example.com /machine PC001 /savefile C:\odj\PC001.txt
ODJ files will be stored in: /srv/odj/
