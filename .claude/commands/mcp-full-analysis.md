description: 全MCP統合でコードベースを完全分析
---

**全7つのMCPサーバ**を統合して、コードベース全体を包括的に分析します。

## 実行内容

このコマンドは以下のMCPサーバを組み合わせて使用します：

### 1. Serena → コード構造解析
- プロジェクト全体のファイル構造
- 主要なクラス・関数の一覧
- モジュール間の依存関係

### 2. Context7 → 技術スタック最新情報
- 使用している各ライブラリの最新ドキュメント
- 推奨パターン・ベストプラクティス
- セキュリティアップデート情報

### 3. Sequential Thinking → アーキテクチャ評価
- 現在の設計の妥当性分析
- 改善提案の段階的導出
- パフォーマンス・セキュリティ観点の評価

### 4. Memory → 過去の設計判断確認
- これまでの技術選定理由
- 過去の問題と解決策
- アーキテクチャ変更履歴

### 5. Filesystem → リソース状況確認
- ディレクトリサイズ
- ファイル数統計
- ストレージ使用状況

## 分析レポート内容

### セクション1: プロジェクト概要
- ファイル数、コード行数
- 主要コンポーネント一覧
- 技術スタック

### セクション2: コード品質
- アーキテクチャパターン
- コーディング規約準拠状況
- テストカバレッジ

### セクション3: 改善提案
- パフォーマンス最適化案
- セキュリティ強化案
- リファクタリング候補

### セクション4: 技術的負債
- 非推奨API使用箇所
- 最新版へのアップグレード推奨
- デッドコード検出

## 使用例

### 完全分析実行
```
/mcp-full-analysis
プロジェクト全体を完全分析し、包括的なレポートを作成してください
```

### 特定モジュール深掘り分析
```
/mcp-full-analysis
flask-appモジュールに絞って詳細分析してください
```

### パフォーマンス重点分析
```
/mcp-full-analysis
パフォーマンス観点を重点的に分析し、ボトルネックと改善案を提示してください
```

全MCPサーバを統合した最も包括的な分析を提供します。
