## 📊 Mermaid図の表示方法

### GitHub/GitLab上で閲覧
- GitHubおよびGitLabはMermaid記法をネイティブサポート
- MarkdownファイルをWebブラウザで開くと自動レンダリング

### ローカル環境
1. **Visual Studio Code + Mermaid拡張機能**
   - 拡張機能: "Markdown Preview Mermaid Support"
   - インストール後、Markdown Previewで図表表示

2. **Typora**
   - Mermaid記法を標準サポート
   - WYSIWYGエディタで快適に編集

3. **Mermaid Live Editor**
   - URL: https://mermaid.live/
   - オンラインでMermaidコードをプレビュー・編集

## 💡 Tips

### Mermaid図のエクスポート

**PNG/SVG形式でエクスポート**:
```bash
# mermaid-cliを使用
npm install -g @mermaid-js/mermaid-cli
mmdc -i システム構成図.md -o システム構成図.png
```

### 図表の印刷

**Markdownを印刷用PDFに変換**:
```bash
# Pandocを使用
pandoc システム構成図.md -o システム構成図.pdf --pdf-engine=wkhtmltopdf
```
