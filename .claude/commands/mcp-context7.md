description: Context7で最新ライブラリドキュメント・コード例を取得
---

**Context7 MCPサーバ**を使用して、最新のライブラリドキュメントとコード例を取得します。

## 使用方法
このコマンドの後に、調べたいライブラリ名やトピックを指定してください。

## 主な用途

### 1. Pythonライブラリ
- **Flask**: 最新のBlueprint、SQLAlchemy統合パターン
- **SQLAlchemy**: マイグレーション、リレーションシップ設計
- **pytest**: テストフィクスチャ、モック使用法
- **requests**: REST APIクライアント実装

### 2. PowerShellモジュール
- **PSWindowsUpdate**: Windows Update自動化
- **ActiveDirectory**: AD操作、ODJ関連
- **Pester**: テストフレームワーク

### 3. JavaScript/フロントエンド
- **Bootstrap**: レスポンシブUI設計
- **jQuery**: DOM操作、AJAX

## 実行内容

Context7のツールを使用して：
1. `mcp__context7__resolve-library-id`: ライブラリIDを解決
2. `mcp__context7__get-library-docs`: 最新ドキュメント取得

## 例

### Flask最新ドキュメント取得
```
/mcp-context7
Flask最新のBlueprint設計パターンとREST API実装例を教えてください
```

### SQLAlchemy マイグレーション
```
/mcp-context7
SQLAlchemy + Alembicでのデータベースマイグレーション手順
```

### PowerShell Windows Update
```
/mcp-context7
PSWindowsUpdateモジュールで完全自動更新する方法
```

Context7から取得した最新情報をもとに、コード例と説明を提供します。
